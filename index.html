Agent Evaluation & Monitoring
Personal Project
Evaluating agentic AI in production is harder than simple Q&A — agents orchestrate multi-step reasoning, tool calls, and feedback loops. Most solutions lock you into a SaaS or cover only a narrow slice.
Designed a 6-layer hybrid evaluation pipeline: instrumentation (LangFuse/OpenTelemetry), trace storage, cost-optimized evaluator cascade (programmatic → LLM-as-Judge generic → domain-specific), test harness with golden datasets, human annotation (Argilla), and Grafana dashboarding with alerting. 100% open-source, zero vendor lock-in.
6-layer pipeline · 7 evaluation dimensions · 100% open-source
LangFuse RAGAS Inspect AI Grafana OpenTelemetry

REMS — RAG Quality Monitor
Personal Project
In regulatory domains, RAG chatbot errors have legal consequences. No lightweight external tool existed to continuously evaluate quality and detect hallucinations — without modifying the chatbot itself.
Built an external evaluation module that plugs into any RAG chatbot via API. Uses RAGAS metrics (faithfulness, context precision, answer relevancy) to detect hallucinations, diagnose root causes, and generate actionable recommendations. Automated weekly evaluations via cron.
Hallucination detection · Automated diagnostics · PDF/HTML reports
